# benn-substack

These are all the posts from [benn.substack.com](https://benn.substack.com). There are markdown versions, batched markdown versions, a CSV, and a JSONL file.

Add more posts by:

1. Exporting the data.
2. Putting what Substack exports into a file called `export-[date]`
3. Running `pip install -r requirements.txt`
4. Running `py scripts/process_export.py export-[date]`

Why? Yeah, I don't know.
